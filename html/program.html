開会式

いよいよ、待ちに待った第92回体育祭『颯』の開幕です。

教室では味わえない、灘校の非日常を、存分にお楽しみください。

閉じる

理事長杯（予選）

体育祭の火蓋を切るのは、中学生による、400m一発勝負の理事長杯予選。

理事長杯の栄光を目指し、まずは学年内の予選を戦います。

閉じる

戦略的棒奪い（予選）

それぞれ得点の異なる棒を、互いの陣地に引き込みあう競技です。

各組の中2・中3が、決勝進出をかけ、白熱した戦いを繰り広げます。

閉じる

棒引き

7本の棒を、自分の陣地に引っ張り合う、中1の学年競技です。

より多くの棒を陣地に入れたほうが勝利となる、シンプルなルールで競います。

中学一年生の初々しさと力強い一面は必見です。

閉じる

N人N+1脚

2人3脚、3人4脚、4人5脚、5人6脚と、徐々に足を繋ぐ人数が増えていきます。

6年間の学校生活で培った強い団結力と協調性で、勝利を目指します。

閉じる

騎馬戦

高2の学年競技として復活を遂げた騎馬戦。

各クラス13騎ずつの騎馬が、敵の大将を討ち取るべく、戦略的でダイナミックな戦を繰り広げます。

閉じる

run for pull

力だけでも技だけでも勝つことが出来ない。それがrun for pullです。

runチームとpullチームの、息の合ったコンビネーションが勝負の鍵を握ります。

閉じる

応援合戦

体育祭に、4クラス対抗の応援合戦が戻ってきました。厳しい練習によって洗練された、個性豊かな演舞をご覧あれ！

※応援合戦については、後ほど特集記事を公開致します。

閉じる

校長杯

昨年から1000mに距離が変更になった校長杯。

俊足を誇る灘高生の電撃の5ハロン戦から目が離せません。

閉じる

THE CHIKARA

最大20キロに及ぶ大量のペットボトルを運ぶ「ワンゲラー」と、さまざまな障害物を超えていく「鉄人レース」の2部構成。

体育祭の中で最も過酷なレースが、灘校生に火を吹きます。

閉じる

相撲

言わずと知れた体育祭の目玉競技。力自慢の生徒たちが、己の最強を証明するべく、土俵に向かいます。

※相撲については、後ほど特集記事を公開致します。

閉じる

戦略的棒奪い（決勝）

予選で組の代表を勝ち取ったチームがトーナメントを組み、優勝を目指して戦います。

戦略が更に洗練され、面白みを増した戦いは見逃せません。

閉じる

ラストラン

名前の通り、高3による10×100mリレーです。

長い灘校生活も残すところ後わずか。灘校生活の集大成と呼べるような、華麗なバトン繋ぎをお見せします。

閉じる

組対抗リレー

各クラスを代表する俊足ランナー達が、組の勝利のために己の力を振り絞ります。

優勝へ学年の壁を越えたリレーをお見逃しなく。

閉じる

理事長杯（決勝）

前もって行われる予選を勝ち上がった強者たちが集います。

勝つのは中3か、それとも下克上があるのか、理事長杯の栄光を目指して400mを走り抜けます。

閉じる

綱引き

体育祭のトリといえばこの競技、毎年恒例の高3全員による綱引きです。

自らの、そしてクラスのプライドを賭けて、最後の戦いに挑みます。

閉じる

理事長杯（決勝）

第92回体育祭『颯』。いかがだったでしょうか。

数年後、数十年後に皆で思い出して笑えるような、そんな記憶の一ページとなれば幸いです。

閉じる
